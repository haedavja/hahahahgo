# CONSTRAINTS — 절대 규칙

## 보안/공개 범위
- `NOTION_TOKEN`은 **코드/커밋/문서에 절대 하드코딩 금지**
- 토큰은 **GitHub Actions Secret**으로만 사용한다(`NOTION_TOKEN`)
- GitHub Pages는 기본적으로 **공개(static) 호스팅**이다  
  → `terminus-graph.json`에 들어가는 내용은 **웹에 공개될 수 있음**  
  → 공개되면 안 되는 설정/스포일러는 넣지 않는다(또는 다른 비공개 호스팅으로 전환)

## 노션 스키마 고정
- DB 속성명은 **한 글자도 바꾸지 않는다**
  - `도메인`, `대분류`, `연관`, `관련 노드`, `상태`
- 타입도 고정(multi_select/select/relation)

## 자동화 동작 원칙
- Pages 배포 워크플로는 **읽기(Export)만** 한다  
  (DB 구조 변경/페이지 생성은 별도 `setup` 명령에서만 수행)
- 실패 시 조용히 넘어가지 말고 **실패로 종료**(Actions 실패 처리)

## UX 원칙(그래프 뷰어)
- 서버/로그인 없이 **정적 페이지로 동작**해야 한다
- 데이터 로딩 실패 시(예: json 없음) **깨지지 말고 안내**해야 한다
- 많은 항목에서도 탐색이 가능하도록
  - 검색/필터/표시 제한/줌 제어를 제공한다
