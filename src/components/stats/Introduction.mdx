# Stats 컴포넌트 라이브러리

통계 및 데이터 표시를 위한 재사용 가능한 컴포넌트 모음입니다.

## 컴포넌트 목록

| 컴포넌트 | 설명 | 사용 예 |
|----------|------|---------|
| **ConfidenceBadge** | 신뢰도 배지 (샘플 크기 기반) | 통계 신뢰도 표시 |
| **WinRateWithCI** | 승률 + 신뢰구간 표시 | 전투 승률 분석 |
| **DataListRow** | 라벨-값 행 | 적별 승률 목록 |
| **SectionTitle** | 섹션 제목 | 카테고리 구분 |
| **StatsGrid** | 그리드 형태 통계 | 요약 통계 |
| **StatsTable** | 테이블 형태 통계 | 상세 데이터 |

## 설계 원칙

### 1. 일관된 스타일링
모든 컴포넌트는 `styles.ts`의 공통 상수를 사용합니다:

```typescript
import { STATS_COLORS, STAT_ROW_STYLE } from './styles';

// 긍정/부정 값 색상
STATS_COLORS.positive  // #22c55e (녹색)
STATS_COLORS.negative  // #ef4444 (빨강)
STATS_COLORS.warning   // #f59e0b (노랑)
```

### 2. 신뢰도 표시
샘플 크기에 따른 통계적 신뢰도를 시각화합니다:

| 샘플 크기 | 신뢰도 | 색상 |
|-----------|--------|------|
| 0-9 | 매우 낮음 | 빨강 |
| 10-29 | 낮음 | 주황 |
| 30-99 | 보통 | 노랑 |
| 100-199 | 높음 | 파랑 |
| 200+ | 매우 높음 | 녹색 |

### 3. 성능 최적화
모든 컴포넌트는 `React.memo`로 감싸져 불필요한 리렌더링을 방지합니다.

## 사용 예시

```tsx
import {
  SectionTitle,
  DataListRow,
  ConfidenceBadge,
  StatsGrid,
  STATS_COLORS,
} from './components/stats';

// 섹션 제목
<SectionTitle emoji="⚔️" color="#fbbf24">
  전투 통계
</SectionTitle>

// 데이터 행
<DataListRow
  label="슬라임"
  value="85.5%"
  valueColor={STATS_COLORS.positive}
  badge={<ConfidenceBadge sampleSize={150} />}
/>

// 그리드 통계
<StatsGrid
  items={[
    { label: '총 전투', value: '150' },
    { label: '승률', value: '75%', valueColor: STATS_COLORS.positive },
  ]}
  columns={2}
/>
```
