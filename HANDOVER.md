## ⚠️ 코드 편집 시 주의사항

### LegacyBattleApp.jsx - HP 바 주변 코드 편집 주의

**문제**: 코드가 매우 커서 HP 바 주변(약 **3700행대**) 수정 시 `apply_patch`가 자주 실패합니다.

**권장 방법**: 해당 구간만 **부분 편집(Edit tool)**을 사용하세요.

**예시**:
```javascript
// ❌ 피하기: 큰 범위 Write/Edit
// 3000~4000줄 전체를 수정하려고 하면 실패 가능

// ✅ 권장: 작은 범위로 나누어 Edit
// HP 바 관련 코드만 정확히 찾아서 수정
```

**HP 바 위치**:
- 플레이어 HP 바: 약 3600~3700줄대
- 적 HP 바: 약 4050~4100줄대

**팁**:
1. Grep으로 정확한 위치 먼저 찾기: `hp-bar-enhanced`, `hp-fill` 등
2. Read로 해당 구간만 읽기 (offset + limit 사용)
3. Edit로 최소 범위만 수정하기
