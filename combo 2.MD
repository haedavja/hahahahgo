1. 디플레이션은 무조건 최종값에서 또 계산한다. 예를들어 50 PT x 4.0 = 200 / 50%(deflation) = 100 이런식이다.
2. 콤보가 주는 기본 PT값은 삭제되고 앞으로 진행단계에서 카드가 진행 순서대로 카드당 10pt를 얻는다. 카드당 얻는 에테르는 추후 변동될 예정.
3. 각 콤보는 다음과 같은 배율을 제공한다. 페어 = 2.0 투페어 = 2.5 트리플 = 3.0 플러쉬 = 3.25 풀하우스 = 3.5 포카드 = 4 파이브카드 = 5
4. 배율을 표시할때, (x 5.0) - 99% 형태로 디플레이션만 %로 표기한다.
