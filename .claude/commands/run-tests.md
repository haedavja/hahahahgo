# 테스트 실행

프로젝트 테스트를 실행합니다.

## 테스트 종류
1. **전체 테스트**: `npm test -- --run`
2. **특정 파일**: `npm test -- --run <파일경로>`
3. **커버리지**: `npm test -- --coverage`

## 주요 테스트 파일
- `src/lib/battleCalculations.test.ts` - 데미지 계산
- `src/components/battle/logic/cardExecutionCore.test.ts` - 카드 실행
- `src/data/reflections.test.ts` - 반영 시스템
