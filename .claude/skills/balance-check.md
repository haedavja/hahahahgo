# 밸런스 체크 스킬

카드, 적, 특성의 게임 밸런스를 분석합니다.

## 사용법
- "밸런스 체크해줘"
- "[카드이름] 밸런스 분석해줘"
- "전체 카드 밸런스 보여줘"

## 분석 항목

### 1. 카드 밸런스
- **효율성 지표**: 피해량 / 속도비용 / 행동력비용
- **비교 분석**: 같은 타입 카드들과 비교
- **특성 시너지**: 특성과의 조합 효과

### 2. 적 밸런스
- **난이도 지표**: 체력 × 평균 피해량
- **턴 예상**: 클리어까지 예상 턴 수
- **위험도**: 플레이어 피해 예상치

### 3. 특성 밸런스
- **가치 평가**: 효과 대비 가중치
- **조합 분석**: 다른 특성과의 시너지

## 분석 공식

### 카드 효율성
```
공격 효율 = 피해량 / (속도비용 + 행동력비용 × 3)
방어 효율 = 방어력 / (속도비용 + 행동력비용 × 3)
```

### 적 난이도
```
난이도 = (적 체력 / 100) × (적 평균 피해량 / 10)
```

## 밸런스 가이드라인

| 등급 | 효율성 범위 | 설명 |
|------|------------|------|
| S | 2.0+ | 너무 강함 (너프 필요) |
| A | 1.5~2.0 | 강함 |
| B | 1.0~1.5 | 적정 |
| C | 0.5~1.0 | 약함 |
| D | 0.5 미만 | 너무 약함 (버프 필요) |
