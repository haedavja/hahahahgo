# 빌드 & 테스트 스킬

프로젝트 빌드와 테스트를 실행합니다.

## 사용법
- "빌드해줘"
- "테스트 돌려줘"
- "빌드하고 테스트해줘"

## 작업 절차

### 1. 빌드 실행
```bash
npm run build
```

### 2. 테스트 실행
```bash
npm test
```

### 3. 개발 서버 실행 (요청 시)
```bash
npm run dev
```

## 오류 처리

### 빌드 오류 시
1. 오류 메시지 분석
2. 자주 발생하는 문제 확인:
   - lucide-react import 오류 → 인라인 SVG로 교체
   - JSX 구조 오류 → wrapper div 확인
   - 타입 오류 → 타입 정의 확인

### 테스트 실패 시
1. 실패한 테스트 케이스 확인
2. 관련 코드 수정
3. 테스트 재실행

## 주의사항
- 코드 변경 후 반드시 빌드 테스트 필수 (CLAUDE.md 규칙)
- 빌드 성공 후 버전 태그 업데이트

- 테스트 도구: Vitest
