# 에이전트 활용 가이드

## 언제 어떤 에이전트를 사용할지

### Explore 에이전트 (코드베이스 탐색)
다음 상황에서 반드시 사용:
- "~가 어디에 있어?" 같은 파일/코드 위치 질문
- "~가 어떻게 동작해?" 같은 구현 이해 질문
- 여러 파일에 걸친 기능 추적
- 동기화 상태 확인 (예: 시뮬레이터 vs 게임)

예시:
```
Task(subagent_type='Explore', prompt='전투 승리 조건이 어떻게 처리되는지 추적해줘')
```

### Plan 에이전트 (구현 계획)
다음 상황에서 사용:
- 새 기능 구현 전 설계
- 리팩토링 계획
- 복잡한 버그 수정 전략

### claude-code-guide 에이전트 (문서 조회)
다음 상황에서 사용:
- Claude Code 기능 질문
- 훅/스킬/커맨드 설정 방법
- Agent SDK 사용법

## 직접 도구 사용 vs 에이전트

| 상황 | 직접 도구 | 에이전트 |
|------|----------|---------|
| 특정 파일 읽기 | Read | - |
| 특정 패턴 검색 | Grep | - |
| 여러 파일 탐색 | - | Explore |
| 코드 흐름 추적 | - | Explore |
| 구현 계획 | - | Plan |
