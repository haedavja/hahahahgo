# 버전 태그 업데이트 스킬

게임 내 맵 하단의 버전 태그를 한국시간 기준으로 갱신합니다.

## 사용법
- "버전 태그 업데이트해줘"
- 코드 작업 완료 후 자동 호출 권장

## 작업 절차

1. **현재 한국 시간 확인**
```bash
TZ='Asia/Seoul' date '+%m-%d %H:%M'
```

2. **버전 태그 위치 찾기**
   - 맵 컴포넌트 하단의 버전 표시 영역

3. **태그 형식**
```
MM-DD HH:MM
예: 01-02 20:45
```

4. **파일 수정**
   - 버전 태그가 있는 컴포넌트 파일 수정

## 버전 태그 찾기

```bash
# 버전 태그 관련 코드 검색
grep -r "버전" src/components/map/
grep -r "version" src/components/map/
```

## 주의사항
- CLAUDE.md 규칙: 작업 후 버전 태그 갱신 필수
- 한국시간(KST, UTC+9) 기준으로 작성
- 24시간 형식 사용 (예: 14:30)
