# 소개

하하하GO는 포커 조합 기반의 턴제 전략 카드 게임입니다.

## 주요 특징

### 전투 시스템
- **타임라인 기반**: 카드 속도에 따른 발동 순서 결정
- **3단계 구조**: 선택 → 대응 → 진행
- **통찰 레벨**: 적 카드 정보 확인 가능 범위 결정

### 성장 시스템
- **에토스 (Ethos)**: 패시브 효과
- **파토스 (Pathos)**: 액티브 스킬
- **로고스 (Logos)**: 궁극기

### 던전 탐험
- **그래프 기반 탐험**: 노드 간 이동
- **다양한 이벤트**: 전투, 보물, 상인, 휴식 등
- **시간 제한**: 제한 시간 내 탈출
