## 🛠️ Unity 포팅 고려사항 (참고용)

### 포팅은 지금 하지 말 것 ❌

**이유:**
1. 웹이 반복 개발 속도가 10배 빠름
2. Unity로 옮기면 신규 기능 추가 속도 급감
3. 아직 "게임이 재미있나?" 단계 - Unity 포팅에 시간 쓰면 본질 잃음

### Unity 포팅 시기
- 웹 버전으로 친구/커뮤니티 피드백 충분히 받은 후
- 게임이 재미있다는 확신이 생긴 후
- 콘텐츠 50% 이상 완성 후

### 대안: 하이브리드 전략
1. **웹 버전**: 무료 베타, 빠른 업데이트, 커뮤니티 피드백
2. **Unity 버전**: Steam 유료 출시, 비주얼 업그레이드, 추가 콘텐츠

## 📝 개발 원칙

### 게임 루프를 한 줄로 설명 가능해야 함
```
전투 → 보상(카드/자원) → 맵 상 이동/선택 → 캐릭터 성장 → 다음 전투
```

### "한 번에 다키스트 던전"이 아니라 "슬더스 미니 버전"을 먼저 완성
- 1~2 캐릭터
- 20~40장 카드
- 10종 내외 적
- 1개 짧은 던전 루프
