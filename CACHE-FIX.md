# 개발 중 브라우저 캐시 비활성화 방법

## Chrome/Edge에서:

1. **F12** 눌러서 개발자 도구 열기
2. **Network** 탭 클릭
3. **Disable cache** 체크박스 체크
4. 개발자 도구를 **열어둔 상태로** 게임 플레이

이렇게 하면 **항상 최신 코드**가 로드됩니다.

## 또는 시크릿 모드 사용:

- **Ctrl + Shift + N** (Chrome/Edge)
- **Ctrl + Shift + P** (Firefox)
- 시크릿 모드는 캐시를 덜 사용합니다

## 강제 새로고침:

- **Ctrl + Shift + R** (Windows/Linux)
- **Cmd + Shift + R** (Mac)
- 캐시를 무시하고 새로 로드합니다
