# 게임 실행 방법

## ⭐ 가장 쉬운 방법: 더블클릭으로 실행

운영체제에 맞는 파일을 **더블클릭**만 하세요:

- **Windows**: `게임실행.bat` 더블클릭
- **Mac/Linux**: `게임실행.sh` 더블클릭

**끝입니다!** 처음 실행할 때도 자동으로 모든 것을 설치합니다.

실행하면:
1. (처음에만) 자동으로 의존성 설치 (1~2분 소요)
2. 개발 서버 시작 (빌드 불필요, 즉시 실행)
3. 브라우저 자동 실행
4. http://localhost:5173 에서 게임 플레이

게임을 종료하려면 열린 터미널/명령창을 닫으면 됩니다.

---

## 📱 아이패드/스마트폰에서 실행하기

컴퓨터에서 게임을 실행한 후, **같은 WiFi에 연결된** 아이패드나 스마트폰에서도 플레이할 수 있습니다!

### 방법:

1. 컴퓨터에서 게임실행 파일을 더블클릭
2. 터미널/명령창에 표시되는 **네트워크 주소**를 확인
   - 예: `http://192.168.0.10:5173` 또는 `http://10.0.0.5:5173`
   - Vite가 자동으로 표시해줍니다
3. 아이패드/폰의 사파리나 크롬에서 그 주소로 접속
4. 게임 플레이!

**주의**:
- 컴퓨터와 아이패드/폰이 **같은 WiFi**에 연결되어 있어야 합니다
- 방화벽이 5173 포트를 차단하고 있다면 허용해야 합니다

---

## 장점

- **빌드 불필요**: 즉시 실행됩니다
- **빠른 시작**: 매번 빌드하지 않아 훨씬 빠릅니다
- **자동 새로고침**: 코드 수정 시 자동으로 반영됩니다
- **404 에러 없음**: Vite 개발 서버가 모든 경로를 처리합니다
- **멀티 디바이스**: 아이패드, 폰 등 여러 기기에서 동시 접속 가능

---

## 명령어로 실행 (개발자용)

### 게임 시작 (개발 모드)
```bash
npm run dev
```
- 가장 빠르고 간단한 방법
- http://localhost:5173 에서 실행
- 코드 수정 시 자동으로 새로고침

### 프로덕션 빌드 후 실행
```bash
npm run build
npm run preview
```
또는
```bash
npm start
```

---

## 명령어 정리

| 명령어 | 설명 |
|--------|------|
| `npm install` | 의존성 설치 (자동 스크립트에서 자동 실행됨) |
| `npm run dev` | 개발 서버 실행 (포트 5173, 추천) |
| `npm start` | 빌드 + 로컬 실행 (포트 3000) |
| `npm run build` | 게임 빌드만 실행 (dist 폴더에 생성) |
| `npm run preview` | 빌드 후 미리보기 (Vite 프리뷰 서버) |

## 문제 해결

### Node.js가 설치되지 않았다는 에러
- https://nodejs.org 에서 Node.js를 다운로드하고 설치하세요
- 설치 후 컴퓨터를 재시작하세요

### 포트가 이미 사용 중이라는 에러
- 5173 포트를 사용하는 다른 프로그램을 종료하거나
- 이전에 실행한 게임 서버를 종료하세요

### 아이패드에서 접속이 안 될 때
- 컴퓨터와 아이패드가 **같은 WiFi**에 연결되어 있는지 확인
- Windows: 방화벽 설정에서 5173 포트 허용
- Mac: 시스템 설정 > 네트워크 > 방화벽에서 포트 허용
- 네트워크 주소가 `http://192.168.x.x:5173` 형태인지 확인 (localhost가 아님)

### Mac에서 "앱을 열 수 없습니다" 에러
터미널에서 다음 명령어 실행:
```bash
chmod +x 게임실행.sh
```
그 후 다시 더블클릭하세요.

### Windows에서 "보안 경고" 메시지
"추가 정보" 클릭 후 "실행"을 선택하세요. 이 스크립트는 안전합니다.
