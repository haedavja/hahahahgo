# 게임 실행 방법

## 로컬에서 게임 실행하기

### 1회만 실행 (의존성 설치)
```bash
npm install
```

### 게임 시작
```bash
npm start
```

이 명령어는:
1. 게임을 빌드하고
2. 로컬 서버를 http://localhost:3000 에서 실행합니다
3. 브라우저에서 http://localhost:3000 을 열면 게임을 플레이할 수 있습니다

### 개발 모드 (코드 수정하면서 테스트)
```bash
npm run dev
```
- 코드 수정 시 자동으로 새로고침됩니다
- http://localhost:5173 에서 실행됩니다

---

## 명령어 정리

| 명령어 | 설명 |
|--------|------|
| `npm install` | 처음 1회만 실행 (의존성 설치) |
| `npm start` | 게임 빌드 + 로컬 실행 (포트 3000) |
| `npm run dev` | 개발 모드 (포트 5173, 자동 새로고침) |
| `npm run build` | 게임 빌드만 실행 (dist 폴더에 생성) |

---

## 문제 해결

### 포트가 이미 사용 중이라는 에러
- 3000 포트를 사용하는 다른 프로그램을 종료하거나
- package.json의 `"start"` 스크립트에서 `-l 3000`을 `-l 3001`로 변경하세요

### 캐시 문제로 변경사항이 안 보일 때
- 브라우저에서 Ctrl+Shift+R (하드 리프레시)
- 또는 시크릿 모드로 실행
