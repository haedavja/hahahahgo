# 게임 실행 방법

## ⭐ 가장 쉬운 방법: 더블클릭으로 실행

### 처음 1회만 (의존성 설치)
터미널이나 명령 프롬프트에서:
```bash
npm install
```

### 게임 실행
운영체제에 맞는 파일을 **더블클릭**하세요:

- **Windows**: `게임실행.bat` 더블클릭
- **Mac/Linux**: `게임실행.sh` 더블클릭

실행하면:
1. 자동으로 게임을 빌드합니다
2. 로컬 서버가 시작됩니다
3. 브라우저가 자동으로 열립니다
4. http://localhost:3000 에서 게임이 실행됩니다

게임을 종료하려면 열린 터미널/명령창을 닫으면 됩니다.

---

## 명령어로 실행 (개발자용)

### 게임 시작
```bash
npm start
```

### 개발 모드 (코드 수정하면서 테스트)
```bash
npm run dev
```
- 코드 수정 시 자동으로 새로고침됩니다
- http://localhost:5173 에서 실행됩니다

---

## 명령어 정리

| 명령어 | 설명 |
|--------|------|
| `npm install` | 처음 1회만 실행 (의존성 설치) |
| `npm start` | 게임 빌드 + 로컬 실행 (포트 3000) |
| `npm run dev` | 개발 모드 (포트 5173, 자동 새로고침) |
| `npm run build` | 게임 빌드만 실행 (dist 폴더에 생성) |

---

## 문제 해결

### npm install이 필요하다는 메시지가 나올 때
```bash
npm install
```
실행 후 다시 게임실행 파일을 더블클릭하세요.

### 포트가 이미 사용 중이라는 에러
- 3000 포트를 사용하는 다른 프로그램을 종료하거나
- 이전에 실행한 게임 서버를 종료하세요

### 캐시 문제로 변경사항이 안 보일 때
- 브라우저에서 Ctrl+Shift+R (하드 리프레시)
- 또는 시크릿 모드로 실행

### Mac에서 "앱을 열 수 없습니다" 에러
터미널에서 다음 명령어 실행:
```bash
chmod +x 게임실행.sh
```
그 후 다시 더블클릭하세요.
