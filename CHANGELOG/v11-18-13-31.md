## 주요 변경사항

### 1. 카드 행동력 텍스트 위치 미세 조정 🎯
**변경**: 카드 왼쪽 상단의 행동력 배지를 1px 아래로 이동

### 2. 한 단계 버튼 단축키 변경 🎮
**변경**: A키 → E키로 단축키 통합

#### 배경:
- E키는 이미 선택/대응 단계에서 "제출" 기능으로 사용 중
- 진행 단계에서는 E키가 "한 단계" 기능으로 작동
- **단일 키로 진행 흐름 통합**: 선택 → 제출(E) → 진행 → 한 단계(E)

#### 단축키 흐름:
```
선택 단계: E키 → 제출 (진행 시작)
    ↓
대응 단계: E키 → 제출 (진행 시작)
    ↓
진행 단계: E키 → 한 단계 실행
```

#### 효과:
- 더 직관적인 조작감
- E키로 게임 진행을 일관되게 제어
- 왼손으로 모든 핵심 동작 가능 (Q, E, R, D, F)

### 3. 콤보 텍스트 크기 추가 30% 증가 📈
**변경**: 콤보 표시가 더욱 크고 강조됩니다.

#### 세부사항:
