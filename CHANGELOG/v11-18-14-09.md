## 주요 변경사항

### 1. 배율 텍스트 크기 40% 감소 📉
**변경**: 콤보 에테르 배율 표시 (×0.50 등)가 더 적절한 크기로 조정되었습니다.

#### 세부사항:
- 이전: `font-size: 1.04em`
- 현재: `font-size: 0.624em` (정확히 40% 감소)
- 에테르 획득량 대비 보조 정보로서 적절한 비율

#### 효과:
- 배율 정보가 과도하게 크지 않게 조정
- 에테르 획득량이 주요 정보로 더 명확하게 강조
- 전체적인 시각적 균형 개선

### 2. 에테르 획득량 텍스트 크기 20% 감소 📊
**변경**: "+15 PT" 등의 에테르 획득량 표시가 적절한 크기로 조정되었습니다.

#### 세부사항:
- 이전: `font-size: 2.4rem` (38.4px)
- 현재: `font-size: 1.92rem` (30.72px, 정확히 20% 감소)
- 여전히 콤보 이름보다 크지만 과도하지 않음

#### 효과:
- 화면 공간 효율성 향상
- 콤보 표시 영역의 전체적인 균형 개선
- 가독성 유지하면서 크기 최적화

### 3. 적 타임라인 애니메이션 구현 🔴
**새 기능**: 적의 타임라인 액션에도 진행 애니메이션이 추가되었습니다.

#### 플레이어와의 차별화:
- **플레이어**: 황금색 발광 (rgba(251, 191, 36))
- **적**: 빨간색 발광 (rgba(239, 68, 68))
- 동일한 펄스 효과 (1.2x → 1.35x → 1.2x)
- 동일한 지속시간 (0.6초)

#### 기술적 구현:
- `.timeline-active.marker-player` - 플레이어 전용 애니메이션
- `.timeline-active.marker-enemy` - 적 전용 애니메이션
- `timelinePulsePlayer` 키프레임 (황금색)
- `timelinePulseEnemy` 키프레임 (빨간색)

#### 시각적 효과:
```
플레이어 액션: 황금색 펄스 + 발광
     VS
적 액션: 빨간색 펄스 + 발광
```

#### 중요:
- **사라지지 않음**: 사용자 요청대로 모든 마커가 계속 표시됨
- **명확한 구분**: 색상으로 즉시 누구의 턴인지 파악 가능

### 4. 콤보 정보 텍스트 간격 축소 🎯
**변경**: 콤보 표시 영역 내 요소들의 간격이 50% 감소했습니다.

#### 세부사항:
- **콤보 이름 ↔ 에테르 획득량**: `marginBottom: 4px` → `2px` (50% 감소)
- **에테르 획득량 ↔ 배율**: `gap: 4px`로 명시적 설정 (flexbox 사용)
- 전체 콤보 정보가 더 컴팩트하게 배치

#### 레이아웃 개선:
- 에테르 정보를 flexbox로 변경하여 정렬 개선
- `display: 'flex', alignItems: 'center', justifyContent: 'center'`
- 에테르 텍스트와 배율을 별도 span으로 분리

#### 효과:
- 콤보 정보가 더 응집된 단위로 보임
- 화면 공간 효율성 증가
- 시각적 그룹핑 강화

## 레이아웃 개선

### 이전 구조:
```jsx
<div>
  콤보 이름
</div>
<div>
  +15 PT (×0.50)
</div>
```

### 현재 구조:
```jsx
<div style={{marginBottom: '2px'}}>

</div>

  <span>+15 PT</span>
  <span>(×0.50)</span>
</div>
```

**개선 효과**: flexbox를 사용하여 정렬이 개선되고, 간격이 명시적으로 제어됨
