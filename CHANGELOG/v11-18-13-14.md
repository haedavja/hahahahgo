# v11-18-13:14 패치 노트

## 주요 변경사항

### 1. 간소화 상태에서 카드 hover 시 텍스트 표시 ✨
**기능**: 간소화 모드(Q)에서도 카드에 마우스를 올리면 설명 텍스트가 표시됩니다.

#### 구현:
- CSS 클래스 `simplified-footer` 추가
- 기본 상태: `visibility: hidden`
- hover 상태: `visibility: visible`
- 간소화 모드로 화면을 깔끔하게 유지하면서, 필요할 때 정보를 확인할 수 있습니다.

#### 코드 변경:
```css
.card-footer.simplified-footer {
  visibility: hidden;
}

.game-card-large:hover .card-footer.simplified-footer {
  visibility: visible;
}
```

### 2. 턴 종료 단축키 F키 추가 🎮
**기능**: 진행 단계에서 F키로 턴을 종료할 수 있습니다.

#### 구현:
- 키보드 핸들러에 F키 추가
- 턴 종료 버튼 텍스트: `⏭️ 턴 종료` → `⏭️ 턴 종료 (F)`
- 진행 단계에서만 작동

### 3. 전투 로그 HTML 렌더링 수정 🐛
**문제**: 에테르 획득 로그에서 HTML 태그가 텍스트로 표시되는 문제
- 예: `에테르 +15 pt <span style="color: #ef4444;">×0.50</span>`

**해결**: `dangerouslySetInnerHTML` 사용하여 HTML 태그 렌더링
- 이제 배수 텍스트가 올바른 색상으로 표시됩니다.
- 감소: 붉은색 (`#ef4444`)
- 향후 증가: 파란색 예정

#### 코드 변경:
```jsx
// 이전
<div>{line}</div>

// 수정
<div dangerouslySetInnerHTML={{ __html: line }}></div>
```

### 4. 콤보 텍스트 크기 확대 📈
**변경**: 콤보 표시 텍스트 크기 증가

#### 세부사항:
- 이전: `font-size: 2.25rem` (36px)
- 현재: `font-size: 2.75rem` (44px)
- **22% 크기 증가**로 가독성 향상
