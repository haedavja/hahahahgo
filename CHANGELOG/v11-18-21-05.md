# v11-18-21:05 패치 노트

## 주요 변경사항

### 1. 콤보 이름 텍스트 크기 증가 📈
**변경**: 콤보 이름 텍스트가 에테르 획득량 텍스트와 동일한 크기로 조정되었습니다.

#### 세부사항:
- 이전: `fontSize: '1.5rem'` (24px)
- 현재: `fontSize: '1.92rem'` (30.72px)
- 에테르 획득량 텍스트와 동일한 크기

#### 효과:
- 콤보 정보 영역 내 모든 텍스트가 균일한 크기
- 시각적 일관성 향상
- 콤보 이름이 더 명확하게 보임

### 2. 에테르-배율 간격 통일 🎯
**변경**: 에테르 획득량과 배율 표시 사이의 간격을 콤보 이름과 에테르 획득량 사이의 간격과 동일하게 조정했습니다.

#### 세부사항:
- 이전: `marginLeft: '-0.05em'` (간격 축소)
- 현재: flexbox `gap: '2px'` (콤보-에테르 간격과 동일)
- 세로 간격(marginBottom: 2px)과 가로 간격(gap: 2px)이 통일

#### 기술적 구현:
```jsx
// 이전
<div style={{fontSize: '1.92rem'}}>
  +{pendingComboEther.gain} PT {pendingComboEther.multiplier < 1 && (
    <span style={{marginLeft: '-0.05em'}}>
      (×{multiplier})
    </span>
  )}
</div>

// 현재
<div style={{fontSize: '1.92rem', display: 'flex', gap: '2px'}}>
  <span>+{pendingComboEther.gain} PT</span>
  {pendingComboEther.multiplier < 1 && (
    <span>(×{multiplier})</span>
  )}
</div>
```

#### 효과:
- 콤보 정보 영역 전체에 일관된 간격 적용
- 세로/가로 간격이 모두 2px로 통일
- 시각적 리듬감 향상

## 콤보 정보 영역 구조

### 레이아웃:
```
콤보 이름 (1.92rem)
    ↓ 2px
에테르 획득 (1.92rem) <--2px--> 배율 (0.624em)
```

### 텍스트 크기:
| 요소 | 크기 | 변경 |
|------|------|------|
| 콤보 이름 | 1.92rem (30.72px) | 1.5rem에서 증가 |
| 에테르 획득량 | 1.92rem (30.72px) | 유지 |
| 배율 표시 | 0.624em (19.2px) | 유지 (상대 크기) |

### 간격:
| 간격 | 값 | 비고 |
|------|-----|------|
| 콤보 ↔ 에테르 | 2px | marginBottom |
| 에테르 ↔ 배율 | 2px | flexbox gap |

**모든 간격이 2px로 통일**되어 시각적 일관성 확보
